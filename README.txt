---------------------------------------------------------------------실행 가이드---------------------------------------------------------------------
1. welcome.jsp를 이용하여 페이지를 시작합니다.
2. 페이지 한가운데 버튼을 클릭하여 로그인을 시도합니다.
tomcat-users.xml에 user정보와 admin 정보가 아래 정보와 맞게 구성되어 있어야 합니다.
---------------------------------------------------------------------2.1 고객의 경우---------------------------------------------------------------------
ID : user
PW : user1234
를 입력하여 로그인을 진행합니다.

2.1.1.ex 메뉴 탭 설명
Home - 초기화면 이동
Product - 종합굿즈 화면 이동
Wish - 장바구니 이동
Logout - 로그아웃 후 초기화면 이동

2.1.1 종합굿즈 화면
굿즈 사진을 클릭하여 개별굿즈 화면으로 이동할 수 있습니다.

2.1.2 개별굿즈 화면
굿즈에 대한 설명과 굿즈 주문, 장바구니, 굿즈 목록 버튼이 구비되어 있습니다.
굿즈 주문 - 장바구니 추가
장바구니 - 장바구니 이동
굿즈 목록 - 종합굿즈 화면 이동

2.1.3 장바구니
굿즈 주문을 통해 추가된 굿즈가 정렬되어 있습니다.
각 굿즈의 수량과 총액이 출력됩니다.
삭제, 주문하기, 쇼핑 계속하기 버튼이 구비되어 있습니다.
작은 삭제 - 개별 굿즈 1종류 삭제
큰 삭제 - 모든 굿즈 일괄 삭제
주문하기 - 배송 정보 이동
쇼핑 계속하기 - 종합굿즈 화면 이동

2.1.4 배송 정보 화면
양식에 맞게 빈칸을 작성하시면 됩니다.
이전, 등록, 취소 버튼이 구비되어 있습니다.
이전 - 이전 화면 이동
등록 - 배송 정보 등록 후 주문 정보 화면 이동
취소 - 주문 취소 화면 이동

2.1.5 주문 정보 화면
입력 받은 배송 정보와 굿즈가 출력되어 있습니다.
이전, 주문완료, 취소 버튼이 구비되어 있습니다.
이전 - 이전 화면 이동
주문완료 - 주문 완료 화면 이동
취소 - 주문 취소 화면 이동

2.1.5 주문 완료 화면
주문 완료 출력과 주문 배송 일시, 주문번호가 출력되어 있습니다.
굿즈 목록 버튼이 구비되어 있습니다.
굿즈 목록 - 종합굿즈 화면 이동

이후 2.1.1과 동일한 순서로 진행할 수 있습니다.

---------------------------------------------------------------------2.2 관리자의 경우---------------------------------------------------------------------
ID : admin
PW : admin1234
를 입력하여 로그인을 진행합니다.

2.2.1.ex 메뉴 탭 설명
Home - 초기화면 이동
Product - 종합굿즈 화면 이동
Register - 굿즈 등록 화면 이동
Edit - 굿즈 편집 화면 이동
Delete - 굿즈 삭제 화면 이동
Logout - 로그아웃 후 초기화면 이동

2.2.1 종합굿즈 화면
굿즈 사진을 클릭하여 개별굿즈 화면으로 이동할 수 있습니다.

2.2.2 개별굿즈 화면
굿즈에 대한 설명과 굿즈 목록 버튼이 구비되어 있습니다.
굿즈 목록 - 종합굿즈 화면 이동

2.2.3 굿즈 등록 화면
양식에 맞게 빈칸을 작성하시면 됩니다.
Korean, English, 등록 버튼이 구비되어 있습니다.
Korean - 한국어 처리
English - 영어 처리
등록 - 굿즈 등록 후 종합 굿즈 화면 이동

2.2.4 굿즈 편집 화면
각 굿즈와 수정 버튼이 구비되어 있습니다.
수정 - 굿즈 수정 화면 이동

2.2.5 굿즈 수정 화면
각 양식 글귀가 작성되어 있습니다.
수정하실 부분을 변경하시면 됩니다.
등록 버튼이 구비되어 있습니다.
등록 - 굿즈 추가 후 편집화면 이동

2.2.6 굿즈 삭제 화면
각 굿즈와 수정 버튼이 구비되어 있습니다.
삭제 - 해당 굿즈 삭제

이후 2.2.1과 동일한 순서로 진행할 수 있습니다.
